Key Innovations

Advanced mathematical frameworks for LLM interpretability

📊

Time Series Analysis

Apply FFT, ACF/PACF, and trajectory dynamics to the residual stream. Measure reasoning persistence with the Hurst exponent (H>0.65 = logical, H<0.5 = hallucination).

🔗

Causal Head Mapping

Vector Autoregression (VAR/VECM) with FDR correction reveals which attention heads causally influence each other. Visualized as real-time heatmaps.

🌐

Topological Analysis

Persistent Homology & Euler Characteristic tracking detect structural anomalies in the reasoning manifold. CPU-parallel computation with zero latency.

🎯

Validity Scoring

Composite score (0-100) combining DTW sensitivity, spectral coherence, topological smoothness, and reasoning indicators. Real-time verdict: STRONG/MODERATE/HALLUCINATION.

🔬

Head Interventions

Knockout experiments reveal which attention heads are "load-bearing" vs redundant. Restoration scoring with KL divergence and mediation analysis.

🧠

Chain-of-Thought Segmentation

Hidden Markov Model (HMM) automatically detects reasoning phases: PROMPT→SETUP→ACTIVE→CONCLUSION. Maps to Betti number dynamics.

How It Works

The complete analysis pipeline

1. INTERCEPTOR captures hidden states during generation [Tokens × 896 dim] ↓ 2. OBSERVER applies SVD compression → [Tokens × 8 components] ↓ 3. Parallel analysis streams: ├─ Time Series: FFT, ACF, Hurst exponent, velocity/acceleration ├─ Causality: VAR fitting, Granger tests, FDR correction, PDC ├─ Topology: Persistent Homology, Betti numbers, Euler χ └─ Interventions: Head knockout, restoration scoring, mediation ↓ 4. SYNTHESIZER combines analyses: • Composite Validity Score (0-100) • Verdict Classification (STRONG/MODERATE/HALLUCINATION) • Mathematical Report (causal_report.md) • 3D Trajectory Visualization (trajectory_3d.html) ↓ 5. DASHBOARD streams results in real-time via WebSocket
